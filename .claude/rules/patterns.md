- **authfail-three-branch-guard**: Strict-server handlers gate on `handlerauth.Require*(...)`, branch on `(fail.Err → deperr.WrapDBIfTransient)` vs `(*OpFail(fail) → 401/403)` vs happy path. Define a per-operation `*Fail` mapper adjacent to the handler. → [authfail-three-branch-guard.md](../skills/patterns/authfail-three-branch-guard.md)
- **deperr-translate-pipeline**: Wrap dep failures with `deperr.Wrap{SMTP,DB,DBIfTransient,OAuthProvider,GitSubprocess}` at call sites; `httperr.WriteFromError` classifies via `errors.Is` and emits typed envelope. → [deperr-translate-pipeline.md](../skills/patterns/deperr-translate-pipeline.md)
- **tx-emit-then-fanout**: Inside `store.WithTx`: domain mutation → `EnsureEventSeqRow` → `AllocateNextSeq` → `InsertEvent`. After commit: `events.Log.FanOut`. Fanout never inside the tx. → [tx-emit-then-fanout.md](../skills/patterns/tx-emit-then-fanout.md)
- **dual-dialect-mirror-queries**: For every table, mirror SQL between `db/queries/sqlite/` and `db/queries/postgres/` with identical query names, columns, and `org_id`/`session_id` scoping. Adapters in `internal/db/store/` translate to the unified `store.Store`. → [dual-dialect-mirror-queries.md](../skills/patterns/dual-dialect-mirror-queries.md)
- **per-package-clock-interface**: Each package defines its own `Clock interface{Now() time.Time}` + `realClock{}` fallback so `*testclock.AdvanceableClock` advances every package without import coupling. → [per-package-clock-interface.md](../skills/patterns/per-package-clock-interface.md)
- **testcontainers-fixture-shape**: Each `tests/e2e/fixtures/<dep>/` package exports a `<Type>` struct with both host-side and container-side address fields plus `Start(ctx, t, Options) *<Type>` that registers `t.Cleanup` and skips on missing image. → [testcontainers-fixture-shape.md](../skills/patterns/testcontainers-fixture-shape.md)
- **wrapper-object-rune-store**: Module-level rune stores in `frontend/src/lib/*.svelte.ts` keep `$state`/`$derived` in private `_`-prefixed vars and expose them via an `export const <name> = { get foo() {...}, mutation() {...} }` plain-object facade — Svelte 5 prohibits exporting raw `$state`/`$derived` from a module. → [wrapper-object-rune-store.md](../skills/patterns/wrapper-object-rune-store.md)
- **snippet-children-component**: Reusable primitives in `frontend/src/lib/components/` take `children: Snippet` via destructured `$props()` with string-literal-union variants; render via `{@render children()}`. → [snippet-children-component.md](../skills/patterns/snippet-children-component.md)
- **spec-driven-event-types**: `docs/openapi.yaml` is the single source of truth for WebSocket event-type strings; `events.AllTypes` must mirror the `EventEnvelope.type` enum exactly; `TestEventTypeConstants_MatchOpenAPIYAML` enforces both directions at CI. → [spec-driven-event-types.md](../skills/patterns/spec-driven-event-types.md)
- **view-state-union-machine**: Screens drive UI state with `type ViewState = 'a' | 'b' | 'c'` + `let viewState = $state<ViewState>(...)` + `{#if viewState === 'a'}` template branching; transitions are direct assignments inside async handlers, and the union's typedef carries a comment block describing the state graph. → [view-state-union-machine.md](../skills/patterns/view-state-union-machine.md)
- **openapi-fetch-middleware-client**: One shared `client = createClient<paths>(...)` in `frontend/src/lib/api/client.ts` with `bearerMiddleware` + `unauthorizedMiddleware` attached via `client.use(...)`; `fetch` passed as a `lateFetch` indirection over `globalThis.fetch`. → [openapi-fetch-middleware-client.md](../skills/patterns/openapi-fetch-middleware-client.md)
- **openapi-fetch-result-branch**: Every `client.GET/POST/...` call destructures `{ data, error }` (optionally `response`), branches on `if (data)` for the happy path and on `error` for typed failure (with an inline cast like `(error as { error?: string })?.error`), reserving `try`/`catch` for transport-level throws. → [openapi-fetch-result-branch.md](../skills/patterns/openapi-fetch-result-branch.md)
- **same-origin-returnto-guard**: User-supplied `return_to` strings (from query, sessionStorage, hash) are validated with `returnTo && returnTo.startsWith('/') && !returnTo.startsWith('//')` before passing to `navigate(...)`; falsy/invalid values fall back to a safe default (`/` or `/login`). → [same-origin-returnto-guard.md](../skills/patterns/same-origin-returnto-guard.md)
- **spa-test-module-mock-barrel**: Each screen/component `*.test.ts` declares a top-of-file `vi.mock(...)` block for every `$lib/` singleton it touches (`auth.svelte`, `router.svelte`, `api/client`, `ws.svelte`), routing each verb through a module-scoped `const mockX = vi.fn()` via `(...args) => mockX(...args)` so the spy survives `vi.mock` hoisting; mutable per-test state uses `let mockY` + getter accessor inside the factory. → [spa-test-module-mock-barrel.md](../skills/patterns/spa-test-module-mock-barrel.md)
- **window-location-defineproperty-stub**: jsdom tests that need to stub `window.location.search`/`.hash`/`.pathname`/`.assign` use `Object.defineProperty(window, 'location', { value: { ...window.location, <field> }, writable: true, configurable: true })`, often wrapped in a per-file `setSearch` / `setHash` helper. → [window-location-defineproperty-stub.md](../skills/patterns/window-location-defineproperty-stub.md)
- **per-instance-factory-rune-store**: Rune stores in `frontend/src/lib/**/*.svelte.ts` needing per-mount isolation use `export function create<Name>(...)` returning a plain-object facade with `$state`/`$derived` in the closure body, instead of module-level singletons. → [per-instance-factory-rune-store.md](../skills/patterns/per-instance-factory-rune-store.md)
- **adapter-wrap-helpers**: Single-row/list adapter methods in `internal/db/store/{sqlite,postgres}_adapter.go` collapse to one line via the package-private generics `wrap1[R,D](row, err, mapErr, convert)` and `wrapList[R,D](rows, err, mapErr, convert)` defined in `wrap.go`. → [adapter-wrap-helpers.md](../skills/patterns/adapter-wrap-helpers.md)
- **strict-server-partial-handler-shim**: Handler test files at `internal/portal/<pkg>/*_test.go` define a `<pkg>OnlyStrict struct { *<pkg>.Handler }` wrapper whose receiver methods `panic("not wired")` for every `openapi.StrictServerInterface` operation owned by another package. → [strict-server-partial-handler-shim.md](../skills/patterns/strict-server-partial-handler-shim.md)
- **playground-activity-reset**: Substantive-action write surfaces on playground sessions (comments, finalize, post-receive) call `store.ResetSessionIdleTimer` after the primary mutation succeeds, guarded by `if orgID == playgroundOrgID && h.PlaygroundIdleTimeout > 0`; failure is `slog.WarnContext`-logged and swallowed. → [playground-activity-reset.md](../skills/patterns/playground-activity-reset.md)
- **ticker-sweep-loop**: Background workers use `ticker := time.NewTicker(d); defer ticker.Stop(); for { select { case <-<stop>: return; case <-ticker.C: doOnePass() } }` where `<stop>` is the goroutine's cancellation channel (ctx.Done, stopCh, or per-handle done). → [ticker-sweep-loop.md](../skills/patterns/ticker-sweep-loop.md)
