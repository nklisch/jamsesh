Geist + Geist Mono (locked)

Modern, neutral, geometric. Vercel's house typeface paired with Geist Mono for code and SHAs. Contemporary without being faddish. Avoids the Inter-default trap — Inter remains in the fallback chain only.

Both faces loaded via Google Fonts. Full system-stack fallback declared so offline rendering degrades cleanly.
