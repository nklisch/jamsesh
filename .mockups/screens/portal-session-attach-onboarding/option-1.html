First-time setup

Three commands, run them in order in Claude Code (the first two are one-time; the third joins this jam).

1

Add the jamsesh marketplace

Tells Claude Code where to find the plugin. One-time per machine.

$ claude plugin marketplace add nklisch/jamsesh
2

Install the plugin

Adds the /jamsesh:* slash commands and the local bin/jamsesh binary.

$ claude plugins install jamsesh
3

Join this jam

Run from a checkout of your source repo. The local binary will OAuth, clone the session, and wire up hooks.

$ /jamsesh:join 01926e42-0000-7000-a000-000000000010

Attach this jam

From a checkout of your source repo:

$ /jamsesh:join 01926e42-0000-7000-a000-000000000010
