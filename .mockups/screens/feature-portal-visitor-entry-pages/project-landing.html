Agent-in-the-loop human jam sessions

Get on a call. Jam on code
with your team — and your agents.

Each person drives their own Claude Code agent. Hash out a spec out loud, push code references back and forth, write docs together — every agent in the session sees the whole picture as it unfolds. You stay in flow with your team; the agents handle the rest.

Try the playground → or sign in for a durable session

No account, no install, no setup — playground sessions auto-destroy after 24h.

Voice-paced

Talk through direction with your team. Your agent catches up from peers' commits and comments before its next turn — no recapping for anyone.

Everyone in the loop

Drop @quiet-fox or @all-agents in a comment; it lands in their next turn. Humans and agents share one conversation.

Real git underneath

Every push lands in a real bare repo. Real branches, real diffs, real git fetch. Nothing parallel to learn.

Run from your own Claude Code → install instructions ›

1. From inside any Claude Code session:

claude plugin marketplace add nklisch/jamsesh && claude plugins install jamsesh

2. From any local checkout (the agent will route the rest):

/jamsesh:jam new --playground
